# Tic Tac Toe
Created Classic **_TicTacToe_** using JavaFX & Kotlin.\
Hope you all enjoy it!

## Aim Of The Game
This is an implementation of the classic game of _**TicTacToe**_\
A 2 player game where, whoever gets their pieces aligned up first wins!
